## 🚀 خطوات التطبيق

### الخطوة 1: تطبيق Migration على Supabase

**الطريقة الأسهل (عبر Dashboard):**

1. اذهب: https://app.supabase.com/project/YOUR_PROJECT_ID
2. **SQL Editor** → جديد
3. انسخ محتوى هذا الملف:
   ```
   supabase/migrations/20260331_fix_expert_pricing_request_id.sql
   ```
4. اضغط **Run**

**أو عبر CLI:**

```bash
supabase db push
```
